# Shared Logic

Эта папка содержит общую логику, которая используется в расширениях для разных IDE.

## Структура

- `services/` - Общие сервисы (ApiService, CacheService)
- `utils/` - Утилиты (key-parser и др.)
- `types/` - Типы и интерфейсы для IDE-специфичных сервисов
- `constants.ts` - Константы

## Использование

Все сервисы в этой папке не зависят от конкретной IDE и используют интерфейсы из `types/` для работы с IDE-специфичными функциями.
