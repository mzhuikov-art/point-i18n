## [Unreleased]

## [1.1.0] - 2024

### Added
- Поиск по переведенному тексту в сайдбаре
  - Поиск файлов по переведенному тексту, даже если в коде используется только ключ i18n
  - Настройка пути поиска (по умолчанию `src`)
  - Автоматическое сохранение последнего пути поиска (`i18nRemote.searchPath`)
  - Отображение результатов с файлом, строкой, ключом и переводом
  - Клик по результату открывает файл на нужной строке
  - Параллельная обработка файлов для ускорения поиска
  - Умная фильтрация исключает ненужные директории (node_modules, dist, build и т.д.)

### Changed
- Оптимизирован поиск по переводам для работы с большими проектами
- Увеличен лимит обрабатываемых файлов с 1000 до 5000
- Улучшена производительность поиска за счет батчевой обработки файлов (по 20 файлов параллельно)
- Использование `readFile` вместо `openTextDocument` для ускорения чтения файлов

## [1.0.0] - 2024

### Added
- Inline-переводы: отображение переводов прямо в коде вместо ключей
- Hover с информацией: просмотр ключа и перевода при наведении курсора
- Быстрое редактирование: клик на перевод для просмотра оригинального ключа
- Синхронизация с сервером: загрузка актуальных переводов из API
- Работа с проектами: выбор проекта и переключение между проектами локализации
- Управление ключами через sidebar:
  - Поиск ключей локализации по названию
  - Создание новых ключей с переводами
  - Редактирование существующих ключей
- Sidebar панель "Управление" в Activity Bar
- Настройки:
  - Язык для отображения переводов (`i18nRemote.locale`)
  - Ключ проекта (`i18nRemote.projectKey`)
- Команды:
  - `Point I18n: Quick Login` - Авторизация и загрузка локалей
  - `Point I18n: Fetch locales now` - Обновить переводы с сервера
  - `Point I18n: Toggle inline translations` - Вкл/выкл inline-переводы
- Горячие клавиши:
  - `Cmd+Shift+I` (macOS) - Включить/выключить inline-переводы
